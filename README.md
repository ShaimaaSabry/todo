# To-Do

A simple To-Do app that can be used to learn the basics of Go.

## Technologies

- Go

## Run Locally

1. Run the Go application:
     ```bash
     go run cmd/main.go 
     ```

## Tests & Linting

* Linting:
    ```
   golangci-lint run --fix
    ```

* Unit tests:
    ```
    go test -cover ./...
    ```
